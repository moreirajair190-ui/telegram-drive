TgPlayer v6.4.15 — Como gerar e distribuir o EXE
==================================================

1) Extraia o projeto em uma pasta simples, por exemplo:

   C:\TgPlayer

2) Instale Python 3.11 ou 3.12 de 64 bits pelo site python.org.
   Na instalação, marque:

   Add Python to PATH

3) Dentro da pasta do projeto, execute:

   build_exe.bat --clean

4) Aguarde. A primeira vez pode demorar porque o script instala PySide6.

5) Ao terminar, use SOMENTE um destes:

   TgPlayer_PORTABLE_PARA_ENVIAR.zip

   ou

   dist\TgPlayer\TgPlayer.exe

6) Para enviar ao usuário final, envie:

   TgPlayer_PORTABLE_PARA_ENVIAR.zip

   ou compacte/envie a pasta inteira:

   dist\TgPlayer

7) NÃO execute e NÃO envie estas pastas:

   build\TgPlayer
   build_intermediario_NAO_EXECUTAR\TgPlayer
   .venv
   .venv-build

Essas pastas são internas/temporárias do build. Se você executar o EXE de dentro delas,
pode aparecer o erro:

   Failed to load Python DLL ... _internal\python311.dll

Esse erro significa que o EXE está sem a pasta _internal ou que o pacote foi copiado incompleto.

8) Se o erro persistir mesmo em dist\TgPlayer:

   - execute build_exe.bat --clean;
   - confira se existe dist\TgPlayer\_internal\python311.dll;
   - veja se o antivírus colocou python311.dll em quarentena;
   - instale Python 3.11/3.12 de 64 bits;
   - envie o arquivo build_log.txt para análise.

9) Build alternativo:

   build_exe_unico.bat --clean

Gera um único arquivo em:

   dist_onefile\TgPlayer.exe

Mas a versão em pasta é recomendada para maior estabilidade.
